# oss-easy

an aliyun oss service client for nodejs, exposes api more like nodejs.fs

## Getting Started
Install the module with: `npm install oss-easy`

```javascript
var oss_easy = require('oss-easy');
oss_easy.awesome(); // "awesome"
```
